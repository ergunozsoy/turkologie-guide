# Turkologie Guide

Ergänzender FAQ-Wegweiser für Studieninteressierte der Turkologie und der Nah- und Mitteloststudien.

## Wichtig

Diese Seite ist keine offizielle LMU-Seite und ersetzt keine offiziellen Informationen des Instituts oder der LMU. Verbindliche Informationen finden sich ausschließlich auf den offiziellen Seiten.

## GitHub Pages aktivieren

1. Repository `turkologie-guide` erstellen.
2. Dateien `index.html`, `style.css`, `script.js`, `README.md` hochladen.
3. Unter **Settings → Pages** gehen.
4. Bei **Source** auswählen: `Deploy from a branch`.
5. Branch: `main`, Folder: `/root`.
6. Speichern.

Die Seite erscheint danach unter:

`https://DEIN-BENUTZERNAME.github.io/turkologie-guide/`
